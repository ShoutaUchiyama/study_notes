## 気づき
- CSSファイルを使用する場合は`<head>`タグに書く。書き方は以下の通り。
```
  <head>
    <title>タイトル</title>
    # ここに記載
    <link rel="stylesheet" href="style.css">
  </head>
```
- id属性は1ページに1つのHTMLタグにしか設定できない。
- 使いまわしたい場合はclass属性を使用する。
- RBGAのAは透明度。0.0（完全透明）から1.0（完全不透明）で設定する。
