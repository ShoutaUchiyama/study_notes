## パーシャル

**パーシャルとは部分テンプレート**
パーシャルではあらかじめ繰り返し使用するHTMLのコードを記述しておき、必要なときにそのHTMLコードを好きな個所に埋め込むことが出来る機能。

**パーシャルの作成のルール**
- パーシャルのファイルの先頭はアンダースコアをつけること
- `views`ディレクトリ内に`shared`ディレクトリを作成し、その中にパーシャルのファイルを記述すること。

**helloのパーシャルファイルを作成する場合の例**
```
_hello.html.erb : 〇 パーシャルとして使用可能
hello.html.erb : × パーシャルとして使用不可（先頭が_ではない）
```

**sharedディレクトリの作成場所**
`app/views/shared`
この`shared`ディレクトリにはパーシャルとして使用するファイルを配置するようにする。

**パーシャルの読み込み方**
`render "ディレクトリ名/パーシャルのファイル名"`

| | helloファイルの場合 | 注意点 |
| :-- | :-- | :-- |
| パーシャルで読み込むファイル | shared/_hello.html.erb | ファイル名の先頭に_を付ける |
| renderで呼び出すとき | <%=render"shared/hello"%> | _と.html.erbは不要 |
